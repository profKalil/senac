Bootstrap : aula introdutória

Conceito

  • Framwork. Funciona como uma "biblioteca" de classes CSS
  • Serve também para Javascript (JS)

Configuração

  • Usar o link direto no caminho da folha de estilo
  • Baixar da internet o ZIP e "linkar" o arquivo do bootstrap
  • Com um gerenciador de pacotes, como npm "npm install bootstrap@versao"

Container

  • Usamos uma DIV com a classe "container" e ela envolve o conteúdo responsivamente
  • Outra opção é usar "container-fluid"

Ajustes do Bootstrap

  • Margem e padding gerais: margem (m) e padding (p)
  • Margem e padding específicos: top (t), right (r), bottom (b), left (l)
  • Medidas: 0 a 5
  • Largura: w. Medidas de 0 a 100
  • Cores: primary(azul), secundary(cinza), success(verde), danger(vermelho), info(azul claro), light (clara), dark (escuro), muted (cinza suave) e white (branco)
  • Bordas. "border-nome-de-uma-cor"
  • Fundo. "bg-nome-da-cor"
  • Texto. "text-nome-da-cor"
  • Active. Quer dizer que está ativado
  • Disabled. É o contrário do active
  • Rounded. Basta usar a classe para arredondar as pontas. Atributos possíveis sm, lg, circle, pill
  • Outline. É um tipo especial de estilização de um botão
  • Classe botão é btn. Pode usar btn btn-primary btn-outline-primary
  • Sombra ou Shadow. Pode ser shadow ou shadow-sm (menor) ou shadow-lg(maior)
